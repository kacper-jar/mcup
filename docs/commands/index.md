# Commands Overview

`mcup` provides a variety of commands to manage Minecraft servers, templates, configuration, and more.

## Available Commands

| Command                   | Description                                                         |
|:--------------------------|:--------------------------------------------------------------------|
| [`server`](server.md)     | Create and list Minecraft servers.                                  |
| [`template`](template.md) | Manage server templates.                                            |
| [`config`](userconfig.md) | Manage user configuration settings.                                 |
| [`update`](update.md)     | Update the internal locker file with the latest server definitions. |
| [`about`](about.md)       | detailed information about the installed `mcup` version.            |

Use `mcup <command> --help` to see detailed help for a specific command.
